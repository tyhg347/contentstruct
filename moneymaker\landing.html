🕸️ ContentStruct API

把任意网页或文本变成干净的结构化数据。
一条 API，搞定内容抓取、清洗、关键词提取。

🚀 开始使用 → 
# curl 示例 — 抓取网页
curl -X POST https://api.contentstruct.dev/extract \
  -H "X-API-Key: cs_your_key" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/article"}'

# 返回
{
  "success": true,
  "title": "文章标题",
  "content": "干净的正文...",
  "word_count": 843
}

pip install requests

import requests

API = "https://api.contentstruct.dev/extract"
resp = requests.post(API, json={
    "url": "https://news.ycombinator.com"
}, headers={"X-API-Key": "cs_your_key"})

data = resp.json()
print(data["title"])      # Hacker News
print(data["content"])    # 干净的页面文本
print(data["word_count"]) # 423
